# Sales Analysis & Forecasting

Sales analysis on a US retail dataset (2015-2018) — EDA, time series forecasting, anomaly detection, and product clustering.

## Files

- `analysis.ipynb` — main notebook 
- `app.py` — Streamlit dashboard
- `summary.docx` — exec summary report
- `train.csv` — raw data
- `charts/` — saved plots

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```
## Stack

pandas, numpy, statsmodels, prophet, xgboost, scikit-learn, streamlit
